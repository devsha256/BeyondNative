Environment Setup Guide

                

                
            

            

                

                    
                    
                    
                


                
                

                    

                        
1. Install Java (JDK 11+)

                        
Recommended: Amazon Corretto 11. Run: brew install --cask corretto

                    

                    

                        
2. Install DataWeave CLI

                        
Option A: Homebrew (Recommended)

                        
brew tap mulesoft-labs/data-weave
brew install dw

                        
Option B: Manual Installation

                        
1. Download the Mac binary from GitHub Releases.

                        
2. Extract to ~/.dw and add its bin folder to your ~/.zshrc:

                        
export PATH=$PATH:~/.dw/bin

                    

                    

                        
3. Verify Installation

                        
Restart terminal and run: dw --version
