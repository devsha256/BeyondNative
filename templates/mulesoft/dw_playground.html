Environment Setup Guide

                

                
            

            

                

                    
                    
                    
                


                
                

                    

                        
1. Install Java (JDK 11+)

                        
Recommended: Amazon Corretto 11. Run: brew install --cask corretto

                    

                    

                        
2. Configure Path

                        
Ensure JAVA_HOME is in ~/.zshrc:

                        
export JAVA_HOME=$(/usr/libexec/java_home)
export PATH=$JAVA_HOME/bin:$PATH

                    

                    

                        
3. Install DataWeave CLI

                        
1. Download the Mac binary from mulesoft/data-weave-cli/releases.

                        
2. Extract to ~/.dw and add its bin folder to your ~/.zshrc:

                        
export PATH=$PATH:~/.dw/bin

                        
3. Restart terminal and run dw --version.

                    

                


                
                

                    

                        
1. Install Java (JDK 11+)

                        
Download the MSI installer for Amazon Corretto 11 or 17.

                    

                    

                        
2. Configure Path

                        
Add JAVA_HOME to System Variables. Then add %JAVA_HOME%\bin to your Path.

                    

                    

                        
3. Install DataWeave CLI

                        
1. Download the Windows binary from mulesoft/data-weave-cli/releases.

                        
2. Extract to C:\dw and add C:\dw\bin to your system PATH.

                        
3. Restart Terminal and run dw --version.

                    

                


                
                

                    

                        
1. Install Java (JDK 11+)

                        
Ubuntu/Debian: sudo apt install openjdk-11-jdk

                    

                    

                        
2. Configure Path

                        
Add to ~/.bashrc:

                        
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH

                    

                    

                        
3. Install DataWeave CLI

                        
Download the Linux binary from mulesoft/data-weave-cli, extract to ~/.dw, and add ~/.dw/bin to your PATH.
